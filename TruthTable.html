For variable names, use alphabets A to Z. Only 4 unique variables can be used (at least for now).

For logical operators, use <!> for NOT, <&> for AND, <|> for OR. Only these three operators are available; if other gates like XNOR, NAND, etc, are needed, conversion is necessary (refer to cheatsheet).

eg. To generate the truth table for
A' OR (B OR C')'
the input ought to be
!A | !(B | !C)

Note: please ensure input is correct as there is no input validation.

Operator Example Converted Form
NAND A NAND B !(A & B)
NOR A NOR B !(A | B)
XOR A XOR B (A | B) & (!A | !B)
XNOR A XNOR B (A & B) | (!A & !B)
